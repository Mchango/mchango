## Tech Stack

**Client:** React, TailwindCSS, Next.JS, Javascript

**Server:** Solidity, Javascript, Ethers.js




## Run Locally

Clone the project

```bash
  git clone https://github.com/Mchango/mchango.git
```

Go to the project directory

```bash
  cd dir
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
