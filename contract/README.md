## Tech Stack

**Server:** `Solidity`, `Javascript`, `Ethers.js`

## Run Locally

Clone the project

```bash
  git clone https://github.com/Mchango/mchango.git
```

Go to the project directory

```bash
  cd contract
```

Install dependencies

```bash
  npm install
```

Compile contract

```bash
  hardhat compile
```
